TS Console Object Documentation Classes
TSDoc Engine Objects
Structured data is focused around a top-level TSDocClass container object. This class holds structures to access field and method definitions along with assoicated inline documentation.
TSDocFieldset: Container object to access lists of TSDocField objects.
TSDocField: Object holding information about a TS class field.
TSDocMethodSet: Container object to access lists of TSDocMethod objects.
TSDocMethod: Object holding information about a TS class method.
TSDocArgSet: Container object to access lists of TSDocArg objects.
TSDocArg: Object holding information about a TS class method argument.
Each of these classes has a script counterpart exposed throught the SysDoc classes.
TSDocClassSet Class
The TSDocClassSet class can build lists of TSDocClass objects and perform group operations. Namely, applying templates. This class is internal and not exposed directly through the SysDoc console classes.
Usage Documentation
Usage documentation is handled through a universal TSDocUsage class. Each class within the TSDocClass structure points to it's own TSDocUsage object that should store related documentation if properly formatted documentation for that element is provided in code comments.
TSDocTemplate Class
The TSDocTemplate class holds formatting strings for building structured output. A TSDocTemplate object has fields to format every TSDoc class and is passed by container objects to children during the applyTemplate() process.
