TS Console Object Documentation Classes
Templating Overview
The templating system can generate flexible output of the TSDoc data. Objects apply a simple macro-replacement to arbitrary text, generally fed by a templating system. Templates are defined in script using a ScriptObject with designated fields defining an output prototype for a TSDoc object.
Templating Fields.
A template can be generated by creating a new Script Object with the following fields:
templateVersion Keep track of future revisions.
templateName Friendly name for the template
formatEOL Will render EOL tokens with a string literal "\n" value.
classTemplate Text body (with macros) to be processed by SysDocClass objects.
classUsageTemplate Text body (with macros) to be generated for usage of SysDocClass objects.
classsetTemplate General structure for class group rendering.
classsetsetEol EOL character to use at the end of class object render.
classsetsetDelim Character to use as separator between class objects in group.
classsetsetPad Prepad string used between class objects for formatting.

Macros and Examples
Macros are defined in the engine by TSDoc classes in the format {{macro}}. A full list of macros for each class are documented in docTemplates.tscript
This file also defines a number of example templates to export object methods as script, JSON, and HTML to demonstrate a range of templating capabilities.
